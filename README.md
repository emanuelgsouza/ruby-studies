# Ruby Studies

An repository of studies using Ruby Programming Language.

## Usage with docker

To execute some script, I preffer to use Docker. In ZSH terminal, execute:

```zsh
docker run --name ruby_exec -v $(pwd):/home --rm -i -t -w=/home ruby:slim ruby imc.rb
```
